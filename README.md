# historiapojazdu.gov.pl

app.py automatycznie wpisuje dane do formularza na stronie [historiapojazdu.gov.pl](https://historiapojazdu.gov.pl/) i szuka prawidłowej wartości „Data pierwszej rejestracji”. Skrypt wymaga w tym samym katalogu pliku konfiguracyjnego 'car_data.py' zawierającego zmienne:
* rejestracja [string]
* vin [string]
* rok_rejestracji [int]

Przykładowy plik konfiguracyjny w repozytorium.
### In english:

app.py automatically enters the data into the form on [historiapojazdu.gov.pl](https://historiapojazdu.gov.pl/) and looks for the correct value of 'Date of first registration'. The script requires a configuration file 'car_data.py' in the same directory containing the variables:
* rejestracja [string].
* vin [string].
* rok_rejestracji [int].

Example configuration file in the repository.
